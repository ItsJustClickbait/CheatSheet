Barebones Of An HTML Document

syntax name description order
<!DOCTYPE html> Declaration The declaration represents what type of document it is and helps the browser to display the page correctly. 1
<html> Tag The HTML tag marks the beginning and end of the html document. All elements in between the tag will be interperited by the browser. 2
<head> Element The element tag contains metadata that isn't typically shown on the webpage. 3
<title> Element The title element sets the title at the top of the web page in the browser (on the browser tab). 4
<body> Element The body element contains all the main information that will be displayed in the webpage. 5
